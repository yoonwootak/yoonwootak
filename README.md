# 백엔드 개발자를 준비 중인 윤우탁입니다.

Spring Boot 기반으로 백엔드 개발을 학습하며  
트랜잭션, 데이터 무결성, 동시성 처리에 관심을 가지고 있습니다.

---

## 🔎 현재 집중하고 있는 것

- 트랜잭션 경계 설정과 동작 원리 이해
- DB 제약 조건을 활용한 데이터 무결성 설계
- 동시 요청 환경에서의 중복 처리 방지

---

## 🛠 기술 스택

Java  
Spring Boot  
Spring Data JPA  
MySQL  
Gradle  
Git / GitHub  

---

## 📌 프로젝트

### seat-reservation-api
티켓팅(좌석 예약) 시스템을 가정한 백엔드 프로젝트

- 동일 좌석 중복 예약 방지
- Service 계층에 트랜잭션 적용
- DB Unique 제약을 통한 2차 방어 설계
- 동시성 처리 방식 실험 예정
